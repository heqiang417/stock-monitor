## 一、策略体系总览

### 1.1 基本面打分模型

所有脚本共享相同的 `fund_score` 函数：

```python
得分 = min(max(roe, 0), 30)              # ROE上限30分
     + min(max(revenue_growth, 0) * 0.4, 20)  # 营收增长×0.4，上限20分
     + min(max(profit_growth, 0) * 0.4, 20)   # 利润增长×0.4，上限20分
     + min(max(gross_margin, 0) * 0.3, 15)     # 毛利率×0.3，上限15分
     + 负债率分 (0-15分)
# 理论满分: 100分
```

硬筛选条件：ROE > 5%，营收增长 > 0%，负债率 < 70%

## 三、核心发现

### 3.1 基本面筛选 + 技术面信号 > 多因子评分

分层筛选优于综合打分。在 PIT TOP200 框架下：

| 因子优先级 | 效果 |
|-----------|------|
| 放量（>1.3x 20日均量） | ✅ 提升最显著 |
| 布林带触底 | ✅ 叠加有效 |
| 股东集中 | ⚠️ 有效但季度数据稀疏 |
| 多因子综合评分 | ❌ 分散信号，效果最差 |

### 3.2 PIT 合规至关重要

旧版用静态基本面分数（最新财报），训练期被系统性地虚高。PIT 合规后：
- 训练期收益普遍下降 5-15%
- 验证/测试集更接近真实表现

### 3.3 已验证的因子效果

| 因子 | 数据量 | PIT可用 | 效果 | 结论 |
|------|--------|---------|------|------|
| RSI | 729万 | ✅ | 有效 | 核心因子 |
| 布林带 | 729万 | ✅ | 有效 | BB触底叠加提升测试 |
| 成交量 | 729万 | ✅ | 有效 | 放量确认显著提升 |
| MACD | 729万 | ✅ | 一般 | 信号太稀疏 |
| 动量 | 729万 | ✅ | 差 | A股超短期动量不延续 |
| 财务基本面 | 12万 | ✅(PIT) | 有效 | TOP200筛选有帮助 |
| 股东集中 | 23万 | ✅(PIT) | 有效 | 测试期提升，但季度稀疏 |
| 主力资金 | 21万 | ✅ | 差 | 仅2000只，覆盖不足 |
| 北向持仓 | 4.3万 | ✅ | 未测试 | 时间太短（2024.1-8） |

## 四、最优策略

### ✅ 唯一通过三阶段 >55% 的策略

**RSI<25 + 放量 + PIT TOP200 + 7天持有**

| 指标 | 训练集 | 验证集 | 测试集 |
|------|--------|--------|--------|
| 正收益率 | 55.3% | 68.1% | 56.7% |
| 交易笔数 | 912 | 207 | 67 |
| 夏普比率 | 0.94 | 2.56 | 1.79 |

**策略逻辑：**
1. 每月初从全市场选出 PIT 基本面评分 TOP200
2. 筛选 RSI < 25 的超卖股票
3. 筛选当日成交量 > 20日均量 × 1.3（放量确认）
4. T+1 开盘买入
5. 持有 7 天后收盘卖出
