## 策略条件

| 条件 | 参数 |
|------|------|
| 布林带 | close ≤ boll_lower × 1.02 |
| RSI | < 20（near_oversold，过滤异常极低值） |
| KDJ | K < 20 **或** J < 0（oversold） |
| 弱市过滤 | width70：当日 ≥70% 个股收盘 < MA20 |
| TOP N | 按 PIT 基本面打分滚动取 TOP500 |
| 买入 | 信号日次一交易日开盘买入 |
| 卖出 | 固定持有7个交易日，于第8个交易日收盘卖出 |
| 成本 | 双边0.30% |

**信号函数**：`signal_rsi_bb_kdj`（RSI + BB + KDJ 联合过滤）
