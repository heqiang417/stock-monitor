> 策略手册 | 候选策略 | 2026-04-03

## 策略条件

| 条件 | 参数 | 说明 |
|------|------|------|
| 大盘弱市 | ≥70% | 当日MA20下方个股比例 |
| 个股RSI | <20 | 14日RSI低于20 |
| 布林带 | close ≤ boll_lower × 0.99 | 价格跌破下轨1%以上（更严格） |
| 基本面 | TOP300 | PIT滚动打分 |
| 放量 | 无 | 不要求放量 |

**全部满足 → 纳入候选 → 按RSI排序取前N → T+1开盘买**

| 阶段 | 胜率 | 夏普 | 索提诺 | 最大回撤 | 盈亏比 |
|------|------|------|--------|---------|--------|
| 训练(熊市) | 61.1% | 1.98 | 4.31 | — | 1.63 |

| 测试(牛市) | 64.5% | 2.67 | 10.57 | 13.7% | 2.54 |

**六项全过 ✅**（胜率>55%、夏普>1.0、索提诺>2.0、最大回撤<50%、盈亏比>1.0、测试笔数>30）

## 与 BB1.00 对比
